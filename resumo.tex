\begin{resumo}
Diversas indústrias, comércios e áreas acadêmicas utilizam bancos de dados para armazenar os mais diversos tipos de dados, gerando uma demanda cada vez maior por métodos de melhoras de perfomance de todos os tipos. Surge, portanto, o desafio de encontrar as melhores otimizações e os melhores gerenciadores de bancos de dados para reduzir o tempo de consulta e volume para cada uso específico.

Ao mesmo tempo, a Teoria de Grafos emerge como um ótimo modelo para resolver, visualizar e armazenar problemas matemáticos, da engenharia, da computação e da indústria pela suas diversas aplicações práticas.

Este trabalho busca estudar e aplicar testes de perfomance nos principais SGDB (Sistemas Gerenciadores de Bancos de Dados) realizando a comparação de perfomance de consulta e volume utilizado para os principais tipos de consultas de bancos de dados de grafos.


\textbf{Palavras-chave} : Base de Dados. Perfomance. Grafos. Sistemas Gerenciadores.
