\section{Objetivo Geral}
O objetivo geral deste trabalho é encontrar o SGBD mais
perfomante para consultas de um banco de dados de grafos
altamente volumoso.

\section{Objetivos Específicos}
\begin{itemize}
      %     \item
      %           Estudar sobre bancos de dados e grafos.
      \item
            Pesquisar e implementar \textit{drivers} para diversos SGBD.
      \item
            Pesquisar e aplicar melhorias de performance para os \textit{drivers} dos SGDB.
      \item
            Aferir e comparar a performance de diferentes SGBD.
      \item
            Estabelecer as vantagens e desvantagens de cada SGBD.
      \item
            Analisar os dados obtidos.
\end{itemize}
